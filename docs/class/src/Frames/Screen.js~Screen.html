    Screen

    Direct Subclass:

    FWSettingsScreen, SlidingScreen, src/X/TestScreen.js~TestScreen, src/X/TestScreen.js~TestScreen2

    Indirect Subclass:

    src/X/TestScreen.js~TestScreen3

    Screen class. This class provides basics of application screen. Do not use directly! Create your own class that extends this and inside him provide manipulations. Example:

    class MyGreatScreen extends Screen {
  onCreate(data) {
    // Do screen filling here!
    this.appendView(new TextView("title", "Hello"))
  }
}

    To start a screen: new MyGreatScreen().start()
